# RAG Chatbot

### Memory-based chatbot with Gemini, FAISS, MongoDB, and React

[![Python](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/)
[![React](https://img.shields.io/badge/frontend-React-61dafb.svg)](https://react.dev/)
[![Flask](https://img.shields.io/badge/backend-Flask-black.svg)](https://flask.palletsprojects.com/)
[![MongoDB](https://img.shields.io/badge/database-MongoDB-47A248.svg)](https://www.mongodb.com/)
[![FAISS](https://img.shields.io/badge/vector%20search-FAISS-0467DF.svg)](https://github.com/facebookresearch/faiss)

A personalized RAG chatbot with memory, built using Gemini for responses, Hugging Face embeddings for retrieval, FAISS for vector search, MongoDB for memory storage, and React for the frontend.

[Features](#features) • [Setup](#setup) • [Usage](#usage) • [How It Works](#how-it-works)

| Feature | Description |
|---------|-------------|
| Chat memory | Stores user-specific summaries in MongoDB |
| Retrieval | Uses FAISS + embeddings to fetch relevant context |
| Gemini responses | Generates answers from context + history |
