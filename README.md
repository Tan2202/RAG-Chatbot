1. Project Overview
Giải thích ngắn đây là gì. Ví dụ:

Laxus RAG Chatbot là một chatbot có bộ nhớ, sử dụng mô hình Gemini để trả lời, FAISS để truy xuất ngữ cảnh, MongoDB để lưu memory người dùng, và React + Vite cho giao diện frontend.

2. Features
Liệt kê ngắn các chức năng chính:

Chat với AI theo ngữ cảnh
Lưu memory theo từng username
Truy xuất dữ liệu bằng FAISS
Tóm tắt hội thoại và lưu vào MongoDB
Frontend React với background media
Backend Flask kết nối Gemini API và Hugging Face embeddings
3. Tech Stack
Viết rõ công nghệ đang dùng:

Frontend: React, Vite, Tailwind CSS
Backend: Flask, Python
Vector Search: FAISS
Database: MongoDB
LLM: Gemini
Embeddings: Hugging Face Inference API
